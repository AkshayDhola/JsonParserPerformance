# Contributing to JsonParserPerformance

Thank you for your interest in contributing to JsonParserPerformance.

## Getting Started

### Setup
```bash
git clone https://github.com/your-username/JsonParserPerformance.git
cd JsonParserPerformance
dotnet restore
dotnet build
dotnet test JsonParserPerformance.UnitTests
```

### Contribution Guidelines

## Reporting Issues

When reporting bugs, please include:

- Sample JSON input
- Proto definition
- Expected vs. actual behavior
- .NET version and OS

## Proposing Features

Describe the use case and proposed solution clearly.

## Submitting Code

1. Fork the repository
2. Create a feature branch (git checkout -b feature/your-feature)
3. Make your changes
4. Run tests: dotnet test JsonParserPerformance.UnitTests
5. Commit with clear messages
6. Push to your fork
7. Open a Pull Request with a detailed description

## Performance Improvements

Include before/after benchmark results using BenchmarkDotNet.

## Questions?

Open an issue for discussion.
